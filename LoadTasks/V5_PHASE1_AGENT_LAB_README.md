# Load Tasks v5 Phase 1

## Added

- Onsite Agent Lab
- Alert Dashboard
- Floating Apple Notes-style Notes system
- Step-by-step How-To help on every major tool
- Dyslexia-friendly testing reports
- Color-coded alert levels with text labels

## Scope

This phase tests and reports while the user is using Load Tasks. It does not claim to run scheduled tests while the browser is closed.

## Alert levels

- Green: Passed
- Yellow: Needs Review
- Orange: Warning
- Red: Broken or Unsafe
- Gray: Not Tested
- Blue: Info

## Locked rule

No build is complete unless verified. Patched items remain Patched, Needs QA until live testing passes.
