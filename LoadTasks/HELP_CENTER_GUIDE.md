# Load Tasks Help Center Guide

## Purpose

The Help Center explains validator warnings, repair order, GitHub Pages problems, iPad Home Screen icon issues, stable build protection, launch readiness, and Focus Mode.

## Load AI Helper

The first version is a local guide. It does not send files or questions to an outside AI service.

Future versions can connect to a private AI provider if the user chooses.

## Main validator rule

A high score with Needs QA means the build looks strong, but warnings still need review.

No build can be marked Complete unless verified.

## Repair order

1. Fix nested folder or GitHub Pages path.
2. Fix duplicate IDs.
3. Review placeholder links.
4. Add CSP.
5. Export Repair Pack.
6. Upload repaired files.
7. Run validator again.
8. Mark stable only after live QA.
