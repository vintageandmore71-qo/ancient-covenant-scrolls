# Load Tasks v5.8 PWA Builder File Inspector

## What this adds

PWA Builder Studio Phase 1 functionality:

- Upload one PWA ZIP
- Inspect package structure
- Count files
- Detect required PWA files
- Detect app root
- Count image, video, audio, and icon files
- Show file tree
- Download inventory report

## What this does not do yet

- No editing
- No repairs
- No rebuild
- No fixed ZIP export from Builder Studio

Those come after this inspector is verified live.
