# Load Tasks v4: Project Intelligence and Build Memory

## What v4 adds

Load Tasks v4 adds project memory, stable build protection, build comparison, stronger promise auditing, false positive detection, launch readiness certificates, and dyslexia-friendly Focus Mode.

## Main rule

No build can be marked Complete unless it is verified.

If a repair is applied but not live tested, its status remains:

Patched, Needs QA

## New sections

- Project Vault
- Stable Build Vault
- Version Timeline
- Build Comparison Intelligence
- Master Build Plan Library
- Promise Audit v2
- False Positive Detector v2
- Launch Readiness Certificate
- Focus Mode

## Best workflow

1. Upload build ZIP.
2. Run Validator.
3. Open Repair Command Center.
4. Apply safe patches only.
5. Open Project Vault.
6. Protect a stable build.
7. Run Promise Audit v2.
8. Generate Launch Readiness Certificate.
9. Upload to GitHub Pages.
10. Test live on iPad.
