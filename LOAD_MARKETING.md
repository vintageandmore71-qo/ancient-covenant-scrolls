Load turns your iPad into an offline launcher for any HTML, PDF, EPUB, or web app. Drop in a file once — read it, edit it, share it forever. No accounts, no subscriptions, no internet required.

## Core promises (use as feature bullets in a store listing)

- **Works fully offline** — once a file is in, you don't need internet again
- **No accounts. No tracking. No subscriptions.** Everything stays on your iPad
- **Built for iPad first** — large tap targets, dyslexia-friendly fonts, no fiddly menus
- **Reads anything** — HTML, PDF, EPUB, DOCX, Markdown, video, audio, images

## What you can do (feature groupings, user-benefit wording)

### Read

- Open PDFs, EPUBs, HTML pages, DOCX files, Markdown, plain text
- Pick up reading right where you left off — every app remembers your place
- Adjust font size, spacing, theme per app (your eyes, your call)
- Dyslexia-friendly font built in (Atkinson Hyperlegible)
- Bionic text overlay — bold the start of each word for faster reading
- Focus line — highlight just the line you're reading
- Bookmarks and notes per app, searchable across your whole library

### Listen

- Read-aloud (TTS) on any text page
- Record your own voice for a chapter and play it back instead of TTS
- Voice Library — save clips and reuse them across the app suite
- Sound Studio — 20 fun voice effects (chipmunk, cartoon, baby, grandma, etc.)
- Voice Manipulator — pitch shift, time stretch, EQ, reverb, echo, presets
- Character Voice Studio — assign different voices to each character in a script

### Edit

- Edit any HTML you import — built-in code editor
- Prose Editor for long-form writing
- Cover Designer for book covers
- Layout View, Chart Maker, Blurb Writer, Metadata Manager
- Save your work as a template, reuse for the next project
- Export everything as standalone HTML

### Create with AI (your key, your provider)

- Built-in AI Helper that explains code, helps with writing, suggests fixes
- Bring your own free API key (Hugging Face, OpenRouter, Gemini, others)
- AI auto-fallback — if one provider can't see images, it switches to one that can
- Coming soon: **Image Prompt** — describe images, generate, refine in chat

### Convert and capture

- Extract audio from any video — save as WAV, M4A, MP3, FLAC, AIFF, AAC, WebM, or OGG
- Save a video → audio extract directly into your library, no detour through Files
- Import multi-file folders by tap-selecting (workaround for iPad's no-folder-picker)

### Organize

- Folders, type filters, full-text search across your library
- ⭐ Favorite the items you want to keep
- 🗑 Clear everything that hasn't been touched in 30+ days (with one consent prompt)
- Add any item to your Home Screen with a custom icon — opens straight into that app

### Share

- AirDrop, Mail, Messages — iPad's native share sheet works on every item
- Save to Files for backup
- Export anything as a single .html file — runs anywhere, no Load required

### Diagnose & fix

- Asset Doctor — scans imported files, tells you what's broken in plain language
- Developer Console — for power users who want to inspect what's happening
- Auto-repair for common iPad-Safari quirks

## Who it's for

- **Dyslexic readers** who want a calmer, font-customizable, distraction-free way to read
- **iPad-only creators** who want desktop-class tools without buying a Mac
- **Offline-first users** — flights, classrooms with bad Wi-Fi, off-grid living
- **Authors & publishers** drafting books, covers, blurbs on iPad
- **Privacy-minded users** who don't want their files in someone else's cloud
- **Educators** delivering content to students without internet at home

---

## Why Load (vs. the alternatives)

- **vs. Safari alone** — Safari can't open most file types, can't run a folder of HTML, can't survive going offline
- **vs. Files app** — Files stores; Load *runs*. Tap a tile, the app opens
- **vs. iCloud / Google Drive** — those need internet. Load doesn't
- **vs. App Store apps** — no review queue, no waiting. Drop in any HTML or PWA, it runs
- **vs. paid PDF readers** — Load reads PDFs *and* EPUBs *and* HTML *and* DOCX *and* media — one app, free

## Honesty notes (so you don't overpromise)

These are real shipped features as of v17cs (2026-04-28):

- Everything in **Read / Listen / Edit / Convert / Organize / Share / Diagnose** above is shipped and working
- **Image Prompt** is in the plan, not yet shipped — say "Coming soon" until it ships
- True voice cloning ("read text in *my* voice") needs an online AI service — we have a plan but it's not built. Don't market this as a current feature yet.
- All AI features need *your* API key — emphasize "bring your own key, free tiers exist" so users aren't surprised
