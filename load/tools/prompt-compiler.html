Compiles a single free-text request into the spec's 11 production layers (visualPrompt, negativePrompt, motionPrompt, cameraPrompt, characterMotionPrompt, environmentPrompt, soundPrompt, musicPrompt, sfxPrompt, voicePrompt, rightsPrompt). Empty layers are returned as '', never undefined. The negative prompt always carries the safety baseline (no text / watermarks / morphing limbs / etc.) plus any "no X" phrases the user typed.
