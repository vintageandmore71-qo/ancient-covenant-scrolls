The Glam-doc plan describes a seven-layer pipeline that any creation request flows through inside Load AI Chat Studio. Below: each layer, what it owns, and the live tool on this build that satisfies it (where one exists). Layers without a live tool are honestly marked not built.

1

Chat input

Free-text user request. Receives the original prose, optional reference image, optional source scene id.
