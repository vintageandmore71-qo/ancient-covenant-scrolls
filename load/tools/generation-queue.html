Generation Queue

BUILD MARKER: GENERATION-QUEUE-v1-2026-05-07-v17g40
Reliability addendum #4. Persistent generation queue. Every image / animate / video / audio / package-export job in Chat Studio writes a record here. Jobs survive page reloads (stored in LoadDB.kv under queue-{id}). Status follows the nine spec values: Queued, Preparing, Generating, Saving, Attaching to scene, Complete, Failed, Cancelled, Retrying. Cancel and retry are real DB writes. No status is set unless the underlying job actually transitioned.
