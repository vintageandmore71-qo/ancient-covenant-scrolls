Run all tests — debug any Load site

Site-wide debugger. For each Load page, runs four real checks: fetch (HTTP 200), frame (loads in iframe), marker (expected build marker / element present), script (no onerror in iframe window). All checks run against the live page on the same origin. Cross-origin pages (the live ACR / Attain hosted apps if checked from elsewhere) will report skip for marker/script.

No false positives: every result is from a real fetch or iframe probe; nothing is hard-coded as PASS.
