How it works

Diagnose every audio layer, separately

If Piper isn't playing, work from top to bottom — Raw audio first, then Browser voice, then Piper Generate, then Piper Playback. Each test reports the exact failure reason. Reset / Repair / Clear-Config buttons live below the diagnostic panel.
