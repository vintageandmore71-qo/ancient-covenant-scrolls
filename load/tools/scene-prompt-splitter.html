Scene Prompt Splitter

X-VIDEO-AI + X-AI-AUDIO Phase 1 — prompt-only

Type a single creation request like “Bring this image to life, have noise and mocking laughter.” The splitter emits the five structured prompts the addendum requires — visual, motion, camera, sound, music, SFX, ambience — and the spec-shaped scene object with all 14 audio fields. Prompt-only by default. No generation provider is called; outputs are scene-ready JSON for later wiring to real engines (ComfyUI, AnimateDiff, FFmpeg.wasm, Load Local Engine).

Output proof rule: this tool only produces prompts; nothing claims a real video or audio file. Generators wired in later phases must verify a real file/blob/URL before marking output ready.
