How it works

Six explicit statuses, one row per provider

Build Plan Part 10 lock: every optional cloud provider reports its true state instead of pretending to work. Test API keys here, and watch for Returned no file — that's when a provider answered without an actual image / audio / file blob, which is when Load must not claim a generation succeeded.

Privacy: The Test buttons send a tiny ping (no user content) to the provider you choose. If you don't tap Test, nothing leaves this device. Keys live in localStorage on this device only.
