Import a PWA

A PWA is a folder of web-app files (HTML + CSS + JS + images, etc.). iPad Safari can't pick a folder directly, so here are the two ways that work:

  1. Easiest — zip the folder first. In the Files app, long-press the folder, tap Compress. That creates a .zip. Then come back here and import that zip.
  2. Or — multi-select every file. Open the folder in Files, tap Select, check every file (index.html plus all assets), and pass them to Load. Load stitches them together automatically.

After import you can View the PWA (default) or Edit its HTML source in a simple text editor, then save your changes back.

Help & FAQ

What is Load, in plain words?

Load is a safe, offline reader on your iPad.

You put files into it — HTML web apps, PDFs, EPUB books. Then you can read them, hear them read aloud, make notes, and bookmark places.

Everything stays on your iPad. Nothing goes to the internet. Nothing is shared.

Is my stuff safe and private?

Yes.

  • Nothing you put in Load is sent anywhere.
  • There is no account, no login, no tracking.
  • Your imported files, notes, and bookmarks live only on your iPad.
  • Load never uses the internet after the first visit.

If you want to save a copy somewhere else, use Settings → Library Backup → Export library to file.

I'm lost. Where am I? How do I go back?

There are four main screens:

  • Home — the big splash picture.
  • Library — the grid of your imported items.
  • Import — the four colored boxes for adding new files.
  • Viewer — where you read one of your items.

The top bar has buttons for each one: 🏠 Home, 📚 Library, ⇪ Import. Tap the one you want.

If you get stuck, just tap 🏠 Home in the top bar. Home is always safe to return to.

What do all the icons in the top bar mean?
  • 🏠 Home — main screen
  • 📚 Library — all your imports
  • ⇪ Import — add new files
  • A− / A+ — make text smaller or bigger
  • ◐ — change the color theme
  • 🌈 — turn OpenDyslexic font on or off
  • 📝 — go to your notes
  • 🔊 — audio / Read Aloud settings
  • ? — this help window
  • ⚙ Aa — all settings

Every icon also has a written label next to it when your iPad is wide enough. On a narrow screen the labels hide to save space.

I can't see the text well. What should I do?

Tap the ⚙ Aa gear icon in the top bar. Try these, one at a time:

  • Tap A+ to make everything bigger.
  • Tap Line spacing ↑ a few times for more room between lines.
  • Tap a color theme: Cream or Sepia is easier on tired eyes. High Contrast is yellow on black — the strongest.
  • Tap Font → OpenDyslexic if letters mix together.

Each change is instant. You can undo anything — tap Reset next to each control.

I can't read for long. Can the app read to me?

Yes. Open any item. In the viewer, tap the 🔊 speaker icon. A small drawer slides up.

  1. Tap Play.
  2. Change Speed if it's too fast or slow.
  3. Try different Voices — ones with a ★ sound the best.

To add more voices, open your iPad's Settings app → Accessibility → Spoken Content → Voices. Pick any voice and tap the download button next to it.

How do I add Load to my iPad home screen?

Make Load itself a home-screen icon so it opens like a native app (no Safari chrome, works offline):

  1. Open Load in Safari (the URL you're using now).
  2. Tap Safari's Share icon (square with an arrow) in the toolbar.
  3. Scroll down the share sheet and tap Add to Home Screen.
  4. Edit the name if you want (default is "Load"), then tap Add.

A Load icon now lives on your iPad home screen. Tap it anytime — it launches full-screen, runs offline, and carries all your imported apps and notes with it.

Tip: adding Load to the home screen once locks in offline mode. Even with Airplane Mode on, tapping the icon opens Load instantly.

How do I add a specific imported app (or book) to my iPad home screen?

You can give every imported file its own dedicated home-screen icon:

  1. Open Library and find the tile you want.
  2. Tap the … button on the tile → Add to Home Screen.
  3. A modal appears with three numbered steps — follow them: tap Safari's Share icon → Add to Home Screen → Add.

The new icon opens Load and jumps straight into that specific app, skipping the home screen. Each app you add this way gets its own distinct icon.

How do I use my own custom icon on the home screen?

iPadOS 15 and newer lets you customize the home-screen icon right in the Add to Home Screen sheet:

  1. Follow the usual steps: tap Safari Share → scroll down → tap Add to Home Screen.
  2. On the next screen, tap the small icon thumbnail at the top-left (next to the name field).
  3. Pick one of:
    • Choose Photo — pick any image from your Photos library.
    • Take Photo — use the iPad camera.
    • Choose File — pick any image file in Files / iCloud Drive.
    • Use Default (Load's violet arrow) — revert to the built-in icon.
  4. Crop / zoom the image if you want. Tap Choose.
  5. Edit the name, then tap Add.

That icon now shows on your iPad home screen. iOS applies its standard rounded-square mask — square or circular images work best, at least 180×180 pixels.

This works for both Load itself and for each per-app home-screen icon you add from a library tile. So every app you add can have its own distinct picture.

Older iPadOS (before 15): iOS uses Load's default icon. Update iPadOS to get custom icon support.

How do I import a file?

Tap Get Started or Import in the topbar. iPad's file picker opens, letting you pick from:

  • On My iPad — files saved locally
  • iCloud Drive — synced across your Apple devices
  • Dropbox, Google Drive, OneDrive, or any other cloud you've signed into in the Files app
  • Recents — files you touched recently
  • Connected drives — USB drives via adapter, external storage

Supported: .html, .zip, .pdf, .epub.

How do I import a PWA?

Tap Import a PWA on the home screen. Two ways that work on iPad:

  1. Zip the folder first. In Files, long-press the folder → Compress. Pick the resulting .zip.
  2. Multi-select files. Open the folder, tap Select, check every file, pass to Load.
How do I use Read Aloud (TTS)?

Open any imported app. In the viewer top bar, tap the 🔊 speaker icon. A drawer slides up with Play/Pause/Stop, speed, and a voice picker.

Enhanced (high-quality) voices are marked with ★ in the dropdown. To install more, open iPad Settings → Accessibility → Spoken Content → Voices and tap the ↓ icon next to any voice to download it.

How do I save my place (bookmarks)?

In the viewer, tap the ★ star icon. Tap "Add bookmark here" and give it a name (like "Chapter 3" or "Page 48"). Later, open the same bookmarks drawer and tap Jump next to any entry to return to that spot.

Your last scroll position is saved automatically even without a named bookmark — Load always re-opens where you left off, and the home screen shows a green "Continue: <app name>" button when you can pick up where you stopped.

How do I use the reading aids?

In the viewer, tap Aa. You get:

  • Font size / line spacing / letter spacing adjusters
  • Color overlays (cream / yellow / soft blue) to cut glare
  • Dyslexia font toggle (OpenDyslexic)
  • Bionic reading (bolds the first ~40% of each word)
  • Focus line (reading ruler)
  • Per-app theme override (stays with just that app)

Global settings live behind the ⚙ gear icon in the topbar.

How do I attach a note to an app?

Two ways:

  1. While viewing — tap the 📝 notes icon in the viewer top bar and write your note.
  2. From the library — tap the … on any tile → Notes.

For a standalone note not tied to any app, tap the 📝 notes icon in the home topbar → + New Note.

Is this really offline?

Yes. Once Load is installed (either opened once in Safari or added to Home Screen), it runs entirely from your iPad. No internet is needed to open Load or any imported file. No data ever leaves your device.

Test it yourself: turn on Airplane Mode, tap the Load icon — it works.

Can I edit an imported app?

Yes. In the library, tap … on any tile → Edit HTML. A full-screen editor opens with the raw HTML. Save writes the changes to Load's local storage; your original file on iPad stays untouched.

My Kindle books won't open

Amazon Kindle files (.azw, .azw3, .mobi, .kfx) are DRM-locked and can only open in Amazon's apps. Convert them to EPUB first using Calibre (free desktop tool, calibre-ebook.com) on a computer, then import the EPUB here.

I deleted something by mistake. Can I get it back?

If you deleted an imported file before exporting a backup, it's gone from Load. But your original file on your iPad is not touched — Load only deleted its own copy.

To find the original file again: open the Files app on your iPad and search for it. Then re-import it into Load.

To protect against this in the future: go to ⚙ Settings → Library Backup → Export library to file. Save the backup to iCloud or email it to yourself once in a while. If anything is lost, import the backup to restore.

I pressed the wrong button. What happens now?

Most buttons do only one small thing and can be undone.

  • Font / theme / spacing changes — tap Reset in that row of the Aa panel.
  • Opened a wrong item — tap the ← back arrow in the top-left of the viewer.
  • Opened a wrong screen — tap 🏠 Home in the top bar.
  • Started typing in the wrong place — tap elsewhere to close the keyboard, nothing is saved.
  • Read Aloud started at the wrong spot — tap Stop in the speaker drawer.

The only buttons that really change things for good are Delete buttons (red). Those always ask "Are you sure?" before anything happens.

The app looks broken / stuck. What do I do?

Try these in order:

  1. Pull down from the top of the page in Safari to refresh.
  2. Close Load (swipe it away from the iPad app switcher) and reopen it from the home-screen icon.
  3. Restart your iPad (hold the power button, slide to power off, then turn it back on).
  4. If Load still looks stuck: open iPad Settings app → Safari → Advanced → Website Data, find the Load entry (dssorit.github.io), swipe left → Delete. Then open Load fresh.

Your imported files, notes, and bookmarks stay on your iPad through all of these steps — unless you choose step 4, which clears everything. So if you ever have to do step 4, export a backup first if you can.

What do my progress, notes, and bookmarks actually save to?

Everything saves to your iPad, inside Safari's storage for Load. Specifically:

  • Imported files (the actual content) go to IndexedDB.
  • Notes and bookmarks attached to each file also live in IndexedDB next to the file.
  • Settings (font, theme, reading aids) live in localStorage.

You don't need to know the technical names. What matters: it all stays on your iPad, nothing is synced to a server, and you can back it up any time with Settings → Library Backup → Export library to file.

My iPad is running low on storage. What can I do?

Big PDFs and EPUBs take up the most space. Each imported file is copied into Load so it works offline.

To free space:

  1. Open Library.
  2. Find items you don't read anymore.
  3. Tap the … on their tiles → Delete.

If you want to keep a backup before deleting, first export: Settings → Library Backup → Export library to file. Then delete from Load. Later you can re-import the backup or just the individual files.

Did my action save? How do I know?

Most changes save automatically the moment you do them.

  • Importing a file: a spinner shows while it works. When the spinner goes away, it's saved.
  • Notes: save when you tap the Save button or when you tap Back.
  • Bookmarks: save the moment you tap "Add bookmark here".
  • Reading position: saves automatically as you scroll — no button to press.
  • Settings: save the moment you tap any option.

Load never loses data from normal use. Data only goes away if you use Delete, or if you clear Safari's website data in your iPad's Settings app.

Short glossary of words Load uses
  • PWA — "Progressive Web App." A web app made of HTML + JavaScript + CSS that can run offline once installed. Load itself is a PWA.
  • EPUB — A free digital book format. Every major non-Amazon book store sells EPUBs.
  • HTML — The format web pages are written in. A single .html file can be a whole small app.
  • ZIP — A folder squished into one file. PWAs that have multiple files are usually shared as a ZIP.
  • TTS / Read Aloud — Text-to-speech. The iPad reads on-screen words aloud with a chosen voice.
  • Bionic reading — A style where the first few letters of each word are bold. Many dyslexic readers find it easier.
  • Focus line — A horizontal highlight that helps your eye follow one line at a time.
  • IndexedDB / localStorage — The iPad's private storage for this app. You don't need to touch these directly.
  • Folder (in Load) — A group you put related items into, like "Books" or "Work." Filter the library by folder with the chips above the grid.
  • Backup — One file that holds all your Load stuff. Use Export to make one, Import to restore it.
Can I upload videos, audio, or photos too?

Yes. On the Import screen (tap Get Started), the orange 🎞 Media card accepts:

  • Video: MP4, MOV, M4V, WebM, MKV
  • Audio: MP3, M4A, WAV, OGG, AAC, FLAC
  • Image: JPG, PNG, GIF, WebP, HEIC, SVG

Each imported file becomes its own tile in the library. Tap a tile and it plays/shows full-screen with native iPad controls (play, pause, scrub for video; zoom for images).

Tip: for long videos, the iPad's built-in Files / Photos app may still be the faster choice. Load is best for short clips you want alongside your readings, books, and notes.

How do I group my stuff into folders or collections?

You can group imported items into folders like "My Books", "Work Tools", or "Reference":

  1. Tap the ⚙ Aa gear in the top bar.
  2. In the Folders section, type a folder name and tap + Add.
  3. Close Settings and open Library.
  4. Tap the … on any tile → Move to folder... → pick a folder by number.

Above the library grid, a row of chips shows All / HTML / PWA / PDFs / Books / Media, plus one chip per folder you've made. Tap any chip to filter. Tap the active chip again to go back to All.

You can rename or delete a folder from the same Settings section. Deleting a folder unassigns its items; it does not delete the items themselves.

How do I back up my whole library?

Load bundles everything — imported apps + books + media, all notes, all bookmarks, folders, your settings — into one file so you can restore later.

  1. Tap ⚙ Aa → scroll to Library Backup.
  2. Tap 📤 Export library to file.
  3. iPad's share sheet opens. Tap Save to Files (or AirDrop it, email it, etc.).
  4. Pick a location you trust — iCloud Drive is a good default.

The file is named load-library-backup-YYYY-MM-DD.json. Do this whenever you add a lot of new items.

How do I restore from a backup?

If you've switched to a new iPad, had to clear Safari data, or want to bring your Load library back:

  1. Tap ⚙ Aa → Library Backup → 📥 Import library from file.
  2. Pick the load-library-backup-*.json file from Files.
  3. Load asks if you want to REPLACE or MERGE:
    • REPLACE — wipes your current Load library and restores the backup exactly.
    • MERGE — adds backup items alongside what you already have. Colliding IDs get a fresh one so nothing overwrites.
  4. Type the word (REPLACE or MERGE) in the box and tap OK. Anything else cancels.
How do I find an item in my library?

Open Library and tap the magnifying-glass (🔍) icon in the top bar.

A search box slides down. Type any word — Load filters the grid live. Search looks inside:

  • Tile names
  • Notes attached to each item

Combine search with the filter chips above the grid (All / HTML / PWA / PDFs / Books / Media / your folders) to narrow further. Tap the ❌ to clear the search.

How does Load remember where I stopped reading?

Load saves your scroll position in any item every 1.5 seconds while you read.

  • Reopen the same item later — you land exactly where you left off.
  • On Home, a green Continue: [name] button appears if any item has a saved position. Tap it to jump back.

For named places like "Chapter 3" or "Page 48", use bookmarks:

  1. In the viewer, scroll to the spot.
  2. Tap the ★ star in the top bar.
  3. Tap + Add bookmark here.
  4. Type a name and tap OK.

Later, tap the same star to see your bookmarks list and tap Jump on any to go back to it.

Can each item have its own look?

Yes. Per-app theme, font size, spacing, and overlay stay with whatever item you last used them on.

In the viewer, tap Aa. In the "This app's theme" row, pick one of Default / Dark / Cream / Sepia / Blue. That choice applies only while you're reading this item. When you leave the viewer, the app-wide theme returns.

Great for: reading books in cream, viewing web apps in dark mode, flipping PDFs in sepia — without constantly reconfiguring.

Where do I see how long I've been reading?

In the viewer, tap Aa. The "Reading time" row shows total minutes spent in that item across all sessions.

No goals, no shaming — it's just a number so you can notice patterns (like "Oh, this book has taken me 3 hours total").

Can I share an item with my family?

Yes — Load lets you export any item as a single self-contained HTML file. Open Library, tap the … on the tile → Share / Export as HTML.

Your iPad's share sheet opens. Pick:

  • AirDrop — send to a nearby family member's iPad / iPhone / Mac. They open the HTML in Safari and it works offline.
  • Mail / Messages — attach the HTML as an email or iMessage.
  • Save to Files — drop it in iCloud Drive, then share the iCloud link.

Whoever receives the file can double-tap it to open in Safari, then tap Share → Add to Home Screen to get their own offline copy.

Notes, bookmarks, and reading position stay with you — only the content itself is exported.

Can I open a website inside Load and use it offline?

Yes, as long as you import the site's files first. A few ways:

  • Single-file HTML sites (like the ACR-Records, ACR-Study, and Attain standalone HTMLs) — import the one .html file and it runs offline forever.
  • Multi-file web apps — zip the whole folder in Files (long-press → Compress), then import the zip. Load inlines every CSS / JS / image / font into one self-contained HTML.
  • Alternative — tap Select in Files, check every file in the folder, and multi-import. Load auto-detects the index.html and bundles the rest.

Once imported, you can: open offline, give it a home-screen icon, export it as a standalone HTML, share it.

Sites that fetch data from a live server won't work offline — those pieces need internet. Pre-bundled static sites work perfectly.

Can Load make a PWA / standalone HTML automatically, without writing code?

Yes. Every imported item in Load already becomes a self-contained HTML — you just need to export it:

  1. Open Library.
  2. Tap the … on any tile → Share / Export as HTML.
  3. iPad's share sheet opens. Save to Files, or AirDrop it.

The exported file is:

  • One HTML file (no external files needed)
  • 100% offline — every CSS / JS / image is inlined
  • Shareable with anyone; they can open it in Safari, Chrome, HTMLq, or add to Home Screen

For a full-library backup (every item in one file), use Settings → Library Backup → Export library to file instead.

Does Load have built-in AI features (summarize, translate, etc.)?

Not yet. Load is currently zero-AI — 100% local, 100% free, 100% offline after install.

Open-source browser-based AI is possible to add later (models like transformers.js can run in Safari), but each model is 30–500 MB extra download, which would break the "stays small and simple" pattern Load has now.

If you want AI help on a specific book or document, two good patterns:

  • Copy a passage, open ChatGPT or Claude in another browser tab, paste.
  • Use iPadOS's built-in Writing Tools (iPadOS 18.1+) — select text, tap Writing Tools, get Summarize / Rewrite / Proofread for free.
Can Load be used as an app tester for developers?

Partly. Load already works as a lightweight sandbox for testing HTML / CSS / JS:

  • Import your build as a single HTML, a ZIP of a folder, or multi-select all the files
  • Tap the tile to run it inside Load's isolated iframe
  • Tap … → Edit HTML to tweak source in-place, then reload
  • The iframe has allow-scripts allow-same-origin allow-forms allow-popups, so most apps run unrestricted

What Load doesn't have: a built-in JavaScript console, network inspector, or live-reload watcher. Safari's Web Inspector (connected from a Mac via USB) gives you those for free on any PWA including Load itself. For iPad-only development, Inspect Browser and Kodex are paid apps that add dev tools.
