Import a PWA

A PWA is a folder of web-app files (HTML + CSS + JS + images, etc.). iPad Safari can't pick a folder directly, so here are the two ways that work:

  1. Easiest — zip the folder first. In the Files app, long-press the folder, tap Compress. That creates a .zip. Then come back here and import that zip.
  2. Or — multi-select every file. Open the folder in Files, tap Select, check every file (index.html plus all assets), and pass them to Load. Load stitches them together automatically.

After import you can View the PWA (default) or Edit its HTML source in a simple text editor, then save your changes back.

Save to Home Screen

Now follow these 3 steps in Safari to save as an icon on your iPad home screen:

  1. Tap the Share icon ⇪ in Safari's toolbar (top right on iPad).
  2. Scroll down the share sheet and tap Add to Home Screen.
  3. Edit the name if you like, then tap Add.

The new icon opens Load and jumps straight into this app. Each app you add this way gets its own icon, just like a native app.
