Auto-Fallback System

Image Prompt automatically routes each request to the best available provider. If your chosen provider can't handle something (e.g. image input), it silently switches to one that can.

Text prompt → Your chosen provider
Image + prompt → Gemini → Anthropic → HF Chat
Image transform → HF SDXL → SD1.5 → FLUX

5. AI Horde — TRULY FREE (no key, no signup) ⭐

Free volunteer GPU pool. Text-to-image only. Slow (~1-3 min per image) but truly unlimited. Anonymous mode used by default — no account required. Last-resort fallback when other providers rate-limit.

deepai.org/dashboard/profile

7. Google Gemini Image (default OFF — opt-in)

Uses your Gemini key. Does both text-to-image AND true pixel-edit via Gemini 2.5 Flash Image — preserves face, pose, and composition very well. Free tier is tight (~500 requests/day, 10/min), easy to burn through during testing. Default OFF. Toggle ON below when you're ready to use it.
