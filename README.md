# Load™ · Attain™ · ACR

Dyslexia-first offline reading + tooling for iPad.

> **Load™** — Run web apps on iPad, fully offline.
> **Attain™** — Structured study and creation tool (commercial release planned).
> **ACR** — Ancient Covenant Record reading + research suite.

---

## Apps in this repository

### Load™
A progressive web app that imports any HTML, PWA, or e-book file and runs it on iPad without a network. Built for readers with dyslexia: OpenDyslexic / Atkinson Hyperlegible fonts, adjustable spacing, text-to-speech, bookmarks, notes, and a built-in Copilot-style AI helper.

**Try it:** [dssorit.github.io/ancient-covenant-scrolls/load/](https://dssorit.github.io/ancient-covenant-scrolls/load/)

### Attain™
A companion app for structured personal study and content creation. Commercial release planned; code visible here during development.

### ACR Reader / Study
Reading and research tools built around the Ancient Covenant Record — original scholarly curation drawing from Dead Sea Scrolls, Orit Ge'ez, and other public-domain primary witnesses. Editorial decisions, translation choices, and commentary are original work.

## Privacy

All apps in this repository are **client-side only**. Nothing the user imports, types, or creates is sent to a server by Load or Attain themselves. Optional AI providers (Gemini, OpenRouter, Hugging Face) are opt-in; when enabled, the user's prompts are sent **directly to that provider using the user's own API key** — they never pass through any server owned by LBond.
